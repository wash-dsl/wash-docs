Detailed Description

Provides declarations for API functions dealing with file IO in the serial mock-up.

Author
James Macer-Wright
Version
0.1
Date
2023-11-15
Copyright

Provides several functions useful for outputting the data to a file periodically, and reading data from those files to continue the simulation from a "checkpoint"

Currently, supports in the implementation HDF5 or ascii (plaintext) file outputs.
