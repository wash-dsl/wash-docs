Introduction

Warwick Smooth Particle Hydrodynamics (WaSH) is a Domain Specific Language used for developing High Performance SPH simulations with built-in parallelisation using OpenMP and CUDA.

This documentation has an Introductory Simulation to show you how to use the WaSH API to write a fully-functional SPH simulation.

Before that, you must make sure all the required libraries and dependencies are installed. Just follow the Installation Steps here.
